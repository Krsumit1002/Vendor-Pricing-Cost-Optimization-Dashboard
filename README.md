# Vendor Pricing & Cost Optimization Dashboard | Excel,Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze vendor pricing, procurement costs, and product price trends. The dashboard helps identify high-cost vendors, monitor procurement efficiency, track price fluctuations, and support data-driven business decisions.

---

## 🚀 Features

* Vendor cost comparison analysis
* Price trend visualization over time
* KPI tracking for procurement insights
* Interactive filters and slicers
* Vendor performance analysis
* Procurement spend distribution analysis
* Multi-page dashboard navigation

---

## 📊 Key KPIs

* Total Spend
* Total Quantity
* Average Price Increase %
* Vendor Performance Rating

---

## 🛠️ Tools & Technologies Used

* Power BI
* Excel
* DAX (Data Analysis Expressions)

---

## 📈 Dashboard Insights

* Identified vendors with highest procurement costs
* Analyzed product price increases over time
* Compared vendor pricing and performance
* Evaluated procurement trends and efficiency

---

## 📂 Dataset

The project uses a simulated business dataset containing:

* Vendor details
* Product pricing
* Procurement quantity
* Lead time
* Vendor ratings
* City-wise procurement information
